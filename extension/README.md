# UXTest Chrome Extension

Widget otomatik olarak her sayfada kalır! Bookmarklet'e gerek yok.

## Kurulum

1. Chrome'da `chrome://extensions` adresine git
2. Sağ üstten **"Developer mode"** (Geliştirici modu) aç
3. **"Load unpacked"** (Paketlenmemiş yükle) tıkla
4. Bu `extension` klasörünü seç
5. Extension yüklendi! ✅

## Kullanım

1. Extension ikonuna tıkla (tarayıcı araç çubuğunda)
2. Listeden bir test seç
3. **"Start"** tıkla
4. Herhangi bir siteye git → Widget otomatik görünür
5. Sayfa değişse bile widget kalır!
6. Test bitince popup'tan **"Stop"** tıkla

## Özellikler

- ✅ Widget her sayfada otomatik yüklenir
- ✅ Sayfa değişince kaybolmaz
- ✅ Session localStorage'da saklanır
- ✅ Backend URL ayarlanabilir
- ✅ Aktif test bilgisi gösterilir
